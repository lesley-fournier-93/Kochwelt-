# 🍳 Kochwelt Recipe Website

A recipe website built with HTML, CSS, and JavaScript.  
This project was created to practice frontend development by building a real-world inspired cooking and recipe platform.

## 🚀 Features
- Browse recipe content in a clean layout
- Responsive design for different screen sizes
- Structured pages with images and text
- Interactive elements powered by JavaScript

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
- `index.html` – Main structure of the website
- `style.css` – Styling and responsive layout
- `script.js` – Interactive functionality

## 📚 What I Learned
- Building structured websites with HTML
- Styling layouts with CSS
- Adding interactivity with JavaScript
- Creating a real-world inspired frontend project

## 🎯 Purpose
This project was built to strengthen my frontend skills and gain hands-on experience with HTML, CSS, and JavaScript.
